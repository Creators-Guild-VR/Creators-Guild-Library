description: Quick image-based guide to add contributors to the library
---


>[!warning]
>Only Admin roles are able to add contributors


1. On GitHub, locate the menu at the top right corner by clicking your avatar image
   ![[Pasted image 20260721041720.png]]
2. Select Organizations
   ![[Pasted image 20260721041804.png]]
3. Click on the Creators Guild
   ![[Pasted image 20260721041920.png]]
4. Click on Teams
   ![[Pasted image 20260721041959.png]]
5. Click on Add a member
   ![[Pasted image 20260721042036.png]]
6. Type out the username, confirm it's the right person using the UI popup, then click invite
   ![[Pasted image 20260721042235.png]]
7. A confirmation message will explicitly say which repositories the user will be able to access and with which roles, if the roles seem alright, just confirm by pressing the green button
   ![[Pasted image 20260721042502.png]]
8. User can now push changes to the library

>[!warning]
>Keep permissions on the minimum needed, otherwise things can get out of control really quick
>
>Most users will only ever need the Write role, anything more than that gives access to changing or even deleting the repositories, Admin is the only role that can add contributors.
