A texture set is a collection of textures that are associated with each other. These textures are combined together to make up a material.

This nomenclature technically comes from Substance Painter, and many artists may use the term interchangeably with material. 

An example texture set may include:
- A base colour
- A roughness
- A metallic map
- A normal map
And when plugged into a shader, these together will give the impression of the material they make up.

Sometimes, textures are combined. Because an image file typically has at least 3 colour channels sometimes multiple values can be combined together into a packed texture.
One example of a packed texture could look like this:

Red Channel - Ambient Occlusion
Green Channel - Roughness
Blue Channel - Metallic
